## 字典数据筛选

> 筛选字典值大于90的数据

```python
from random import randint
d = {x:randint(60,100) for x in range(21)}
print(d)
d = {k:v for k,v in d.items() if v>=90}
print(d)
```

## 集合数据筛选

```python
from random import randint
data = [randint(-10,10) for _ in range(10)]
s = set(data)
s = {x for x in data if x>=0}
print(s)
```

## 字典成绩排序

```python
from random import randint
d = {x:randint(60,100) for x in 'zxcvbned'}
print(d)
d = sorted(d.items(),key=lambda x:x[1])
print(dict(d))
```

## 找到字典中公共key

```python
from random import randint,sample
d1 = {x:randint(1,4) for x in sample('abcdefg',randint(3,6))}
d2 = {x:randint(1,4) for x in sample('abcdefg',randint(3,6))}
d3 = {x:randint(1,4) for x in sample('abcdefg',randint(3,6))}
pub = d1.keys() & d2.keys() & d3.keys()
print(d1)
print(d2)
print(d3)
print(pub)
```
